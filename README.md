# E-commerce Sales Analysis Project

## Project Goal
To analyze e-commerce transaction data to identify sales trends, top products, and regional performance.

## Pipeline
1. **Load:** Read data from CSV.
2. **Clean:** Convert dates and format columns.
3. **Analyze:** Group data by product, month, and region.
4. **Visualize:** Generate Bar, Line, and Pie charts using Matplotlib.

## Insights
- **Top Product:** Laptops are the highest revenue generator.
- **Trend:** There was a significant peak in sales during March.
- **Market:** The North region is the strongest performing territory.
